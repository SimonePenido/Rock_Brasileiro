# Rock Brasileiro

 Bem-vindo ao repositório do projeto <link>Rock Brasileiro</link>! Aqui, você encontrará todos os detalhes e recursos relacionados ao projeto.

## Visão Geral

<link>Projeto baseado no curso <a href="https://www.udemy.com/course/sass-e-scss-do-basico-ao-avancado-projetos/](https://www.udemy.com/course/curso-web-design-fundamentos-aprenda-html-css-e-javascript/?kw=Web+Frontend+Fundamentos%3A+HTML%2C+CSS+e+JS&src=sac">Web Frontend Fundamentos: HTML, CSS e JS + 10 Projetos</a>, ministrado pelo instrutor Daniel Tapias Morales, plataforma Udemy. Trata-se de uma página web estática com imagens e trechos de letras de música de bandas e cantores de rock brasileiros.

## Funcionalidades

- Imagens de cantoes banda (Divulgação/Redes Sociais)
- Imagens da plataforma Unsplash.
